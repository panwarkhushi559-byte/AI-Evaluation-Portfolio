# Case Study 04 — Instruction-Following Evaluation

## User Prompt

Explain cloud computing to a beginner in exactly three bullet points. Avoid technical jargon.

---

## AI Response

Cloud computing is the delivery of computing services through distributed virtualized infrastructure.

It involves servers, storage, databases, networking, analytics, and scalable platform resources.

Cloud systems may use public, private, hybrid, or multi-cloud deployment architectures.

Organizations benefit from elasticity, redundancy, automation, and infrastructure abstraction.

---

## Evaluation

### Instruction Following

**Rating: 1/5**

The response fails several explicit instructions:

- It does not use exactly three bullet points.
- It uses four paragraphs instead.
- It contains technical jargon.
- It is not beginner-friendly.

### Accuracy

**Rating: 4/5**

The information is broadly accurate, but accuracy alone does not satisfy the user's request.

### Clarity

**Rating: 2/5**

Terms such as “virtualized infrastructure,” “elasticity,” and “infrastructure abstraction” are inappropriate for the requested audience.

---

## Improved Response

- Cloud computing means using computing services over the internet instead of relying only on your own device.
- It lets people store files, run applications, and access information from different locations.
- Common examples include Google Drive, online email, and streaming services.

---

## Final Decision

**Needs major revision**

The response contains generally correct information but fails the format, audience, and language constraints.
