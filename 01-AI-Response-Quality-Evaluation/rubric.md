# AI Response Evaluation Rubric

This rubric provides a consistent scoring system for evaluating AI-generated responses.

Each criterion is scored from 0 to 5.

## Accuracy

| Score | Standard |
|---|---|
| 0 | Completely incorrect or misleading |
| 1 | Mostly incorrect with serious factual problems |
| 2 | Contains several significant inaccuracies |
| 3 | Mostly correct but contains noticeable errors |
| 4 | Correct with only minor inaccuracies |
| 5 | Fully accurate and appropriately qualified |

## Completeness

| Score | Standard |
|---|---|
| 0 | Does not answer the request |
| 1 | Addresses only a very small portion |
| 2 | Misses several important requirements |
| 3 | Covers the main request but omits useful details |
| 4 | Covers almost all important requirements |
| 5 | Fully addresses the request without unnecessary content |

## Clarity

| Score | Standard |
|---|---|
| 0 | Unreadable or incoherent |
| 1 | Extremely confusing |
| 2 | Difficult to follow |
| 3 | Understandable but poorly organised |
| 4 | Clear and logically structured |
| 5 | Exceptionally clear, concise, and well organised |

## Reasoning

| Score | Standard |
|---|---|
| 0 | No meaningful reasoning |
| 1 | Conclusions are largely unsupported |
| 2 | Weak or inconsistent reasoning |
| 3 | Generally logical with some gaps |
| 4 | Strong and well-supported reasoning |
| 5 | Rigorous, transparent, and fully supported reasoning |

## Instruction Following

| Score | Standard |
|---|---|
| 0 | Ignores the request |
| 1 | Violates most important instructions |
| 2 | Follows some instructions but misses major requirements |
| 3 | Follows the main request with noticeable omissions |
| 4 | Follows nearly all instructions |
| 5 | Fully satisfies all explicit and relevant implicit instructions |

## Hallucination Control

| Score | Standard |
|---|---|
| 0 | Contains extensive fabricated information |
| 1 | Contains several serious unsupported claims |
| 2 | Contains multiple questionable claims |
| 3 | Mostly grounded but includes limited unsupported content |
| 4 | Well grounded with minor qualification issues |
| 5 | Contains no identified fabricated or unsupported claims |

## Helpfulness

| Score | Standard |
|---|---|
| 0 | Not useful |
| 1 | Provides almost no practical value |
| 2 | Provides limited assistance |
| 3 | Useful but incomplete |
| 4 | Helpful and actionable |
| 5 | Highly useful, precise, and appropriately tailored |
