# AI Coding Evaluation Rubric

Each coding response is evaluated using seven dimensions.

| Criterion | Score |
|-----------|------:|
| Correctness | 0–5 |
| Instruction Following | 0–5 |
| Readability | 0–5 |
| Debugging Quality | 0–5 |
| Edge Cases | 0–5 |
| Explanation | 0–5 |
| Overall Reliability | 0–5 |

Maximum Score: 35

Decision Levels

31–35 → Accept

25–30 → Minor Revision

17–24 → Major Revision

0–16 → Reject
