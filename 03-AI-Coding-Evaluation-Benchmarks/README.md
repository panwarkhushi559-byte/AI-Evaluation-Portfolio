# AI Coding Evaluation Benchmarks

This repository contains structured benchmark tasks designed to evaluate the capabilities and limitations of AI coding assistants.

The focus is not on building software, but on evaluating whether AI-generated code is correct, reliable, maintainable, and aligned with user requirements.

---

# Objective

Develop repeatable benchmark tasks that assess AI coding performance across common software engineering scenarios.

---

# Evaluation Areas

- Code Correctness
- Instruction Following
- Bug Detection
- Debugging Quality
- Code Readability
- Error Analysis
- Edge Case Handling
- Explanation Quality
